# TP 2 : Gestion de l’état local et interactions utilisateurs avec React

## 📚 Cours
Développement Front-End moderne avec React

---

## 🎯 Objectif

Ce TP a pour objectif de comprendre et appliquer les notions suivantes :

- Gestion de l’état local avec `useState`
- Gestion des événements (`onClick`, `onChange`, `onSubmit`)
- Création de formulaires contrôlés
- Utilisation du hook `useEffect`
- Respect du flux unidirectionnel des données

---

## ⚙️ Étapes réalisées

### ✅ Étape 1 : Préparation du projet
- Création du projet avec `create-react-app`
- Vérification du bon fonctionnement

---

### 🔢 Étape 2 : Composant Compteur
- Création de `Compteur.js`
- Utilisation de `useState`
- Incrémentation d’un compteur avec un bouton

---

### 🖱️ Étape 3 : Message avec bouton
- Création de `MessageBouton.js`
- Changement dynamique de texte avec `onClick`

---

### 📝 Étape 4 : Formulaire contrôlé
- Création de `FormulaireNom.js`
- Gestion des entrées utilisateur avec `onChange`
- Soumission du formulaire avec `onSubmit`
- Affichage des données avec une alerte

---

### 🔄 Étape 5 : useEffect
- Création de `CompteurEffet.js`
- Utilisation de `useEffect`
- Affichage des changements dans la console

---

### 🧪 Étape 6 : Exercices pratiques

#### Exercice 1 : Affichage dynamique
- Texte change à chaque clic

#### Exercice 2 : Formulaire d’inscription
- Champs : prénom et email
- Affichage des données dans une alerte

#### Exercice 3 : Compteur avec effet
- Mise à jour du `document.title`
- Affichage du nombre de clics

---
